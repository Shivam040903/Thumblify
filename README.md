# 🚀 Thumblify — AI Thumbnail Generator

Thumblify is a full-stack AI-powered web application that generates high-quality, click-worthy thumbnails using modern AI models.

---

## ✨ Features

* 🎨 AI-powered thumbnail generation
* 🧠 Smart prompt-based customization (style, color, aspect ratio)
* 🔐 Secure authentication with session-based login
* ☁️ Cloudinary integration for image storage
* 📂 User dashboard to view generated thumbnails
* ⬇️ Download thumbnails instantly
* 🗑 Delete thumbnails

---

## 🛠 Tech Stack

### Frontend

* React.js (Vite)
* Axios
* Tailwind CSS

### Backend

* Node.js
* Express.js
* TypeScript
* MongoDB + Mongoose
* Express Session + MongoStore

### AI & Media

* Stability AI (Stable Diffusion API)
* Cloudinary
